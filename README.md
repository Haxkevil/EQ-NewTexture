Para quem não sabe usar:

Use o explorador de arquivos MT Manager.

1. Toque e segure (clique longo) no APK.
2. Selecione Extract to.
3. Toque em OK e aguarde a extração.
4. Vá até a pasta dos arquivos extraídos: assets > data.
5. Faça as substituições necessárias.
6. Volte para a pasta principal: Exiled-Kingdoms-RPG.
7. Toque nos três pontinhos no canto superior direito.
8. Escolha Selecionar tudo.
9. Segure em qualquer arquivo azul; um menu aparecerá. Escolha Comprimir.
10. Importante: no menu Level, selecione APK Mode. Renomeie a extensão do arquivo para .apk.
11. Toque e segure no APK gerado, vá em Sign file e depois OK.
12. Instale o arquivo assinado (ex: Exiled-Kingdoms-RPG_sign).
