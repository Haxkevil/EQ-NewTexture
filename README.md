## Tutorial de Instalação

### Pré-requisito
- Ter o **MT Manager** instalado no dispositivo.

### Passo a passo

1. **Toque e segure** (clique longo) no arquivo APK.
2. Selecione a opção **`Extract to`**.
3. Toque em **OK** e aguarde a extração terminar.
4. Navegue até a pasta extraída:  
   📁 `assets` → `data`
5. Faça as substituições necessárias com os arquivos da textura.
6. Volte para a pasta principal: **`Exiled-Kingdoms-RPG`**
7. Toque nos **três pontinhos** (⋮) no canto superior direito.
8. Selecione **`Selecionar tudo`**.
9. Segure em qualquer arquivo azul — um menu aparecerá.  
   Escolha a opção **`Comprimir`**.
10. ⚠️ **Importante:**  
    No campo *Level*, selecione **`APK Mode`**.  
    Renomeie a extensão do arquivo para **`.apk`**.
11. Toque e segure no APK gerado.  
    Vá em **`Sign file`** → **OK**.
12. Pronto! Instale o arquivo assinado:  
     `Exiled-Kingdoms-RPG_sign.apk`
